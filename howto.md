# How To

This is a guide on how to use the Website www.bearcat-battalion.org.

### Navigation Bar

The navigation bar at the top of the screen allows you to navigate to different pages of the website such as Home, About, and Branches.

### Home Page

The homepage contains general information on ROTC and the University of Cincinnati's program.

### About Page

The about page serves as a resource for future cadets to learn about ROTC, familiarize themselves the Cadre members, learn about summer opportunities such as Airborne, Air assault, CTLT, and Project go. They can also find information about our ranger challenge team.

### Branches Page

Individuals can use the branches page to learn about branches, and they have the ability to filter them by branch type.
